# Simulation de Planétarium (Méthode d'Euler asymétrique)

Ce projet scolaire réalise une simulation du système solaire interne (Mercure, Vénus, Terre, Mars) en utilisant Python.

## 🚀 Fonctionnalités
- Comparaison entre la **méthode d'Euler classique** et la **méthode d'Euler asymétrique**.
- Analyse de la conservation de l'énergie mécanique.
- Animation interactive des orbites avec `Matplotlib`.

## 📈 Résultats
Le projet démontre que la méthode d'Euler asymétrique est plus stable pour les orbites planétaires sur de longues durées (conservation de l'énergie).
